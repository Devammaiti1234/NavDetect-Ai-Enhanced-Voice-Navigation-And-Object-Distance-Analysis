# NavDetect-Ai-Enhanced-Voice-Navigation-And-Object-Distance-Analysis

 Developed an AI-powered voice navigation and object distance analysis system.
  Implemented real-time object detection using computer vision techniques.
 Calculated the distance between the user and detected objects.
  Provided voice-based alerts and navigation guidance for improved accessibility.
 Integrated object detection, distance estimation, and audio feedback into a single system.
 Enhanced spatial awareness by helping users identify nearby obstacles.
 Improved navigation safety through real-time monitoring and voice assistance.
